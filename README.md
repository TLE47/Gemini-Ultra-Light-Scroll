## Gemini Ultra-Light Scroll

A minimalist Tampermonkey script that keeps the Google Gemini chat pinned to the bottom only when the latest message contains a code block.
#
🛠 Installation
1. Open your Tampermonkey dashboard.
2. Create a New User Script.
3. Paste the code and Save (Ctrl+S).
4. Refresh your Gemini tab.
#
📝 Technical Note

This script targets the infinite-scroller and message-content elements. Because it relies on Gemini's specific web structure, it may require updates if Google changes their site layout.
